## Baixando bibliotecas para o projeto
- Inicializar o git
  ```
    git init
  ```
- Inicializar o npm
  ```
    npm init -y
  ```

- Express.js:

  Framework web para Node.js que simplifica a criação de APIs RESTful e rotas.
  ```
  npm install express
  ```
- Mongoose:
  ```
  npm install mongoose
  ```
- dotenv
  ```
  npm install dotenv
  ```
